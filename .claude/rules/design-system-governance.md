# Rule: Design system governance

1. No hardcoded color / spacing / radius / shadow / motion in components or pages.
2. Add to `src/shared/design-system/tokens/` and mirror in `src/styles/theme.css`.
3. Reusable visual components live in `src/shared/design-system/components/` or `src/shared/ui/`.
4. New visual patterns require updating `docs/branding/design-governance.md`.

Reference: docs/branding/design-governance.md.
