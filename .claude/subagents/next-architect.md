# Subagent · next-architect

Reviews App Router structure, server/client boundaries, route groups, layouts, caching, rendering decisions.

Inputs: PRs / branches that touch routing, layouts, server components, caching, proxy or instrumentation.
Output: pass / fail report with specific files and lines, and a fix list.
References: ADR-001..003, `node_modules/next/dist/docs/01-app/`.
