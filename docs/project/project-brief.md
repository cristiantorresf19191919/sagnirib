# Biringas — project brief (resumido)

## Producto
Marketplace donde acompañantes ofrecen servicios para eventos, viajes y compañía. Modelo conceptual cercano a fotoprepago / milerótico, restringido a servicios de acompañamiento (no contenido).

## Acción principal (CTA)
**Compra directa** desde el listado/detalle: el usuario elige una acompañante y la contrata desde la página.

## Páginas iniciales
- Home.
- Listado de servicios (catálogo de acompañantes disponibles).
- Detalle de cada acompañante con CTA de contratación.
- Legales (TOS / privacidad) — pendientes.

## Idiomas
Español (default) e inglés. Routing `[lang]` no activado en foundation.

## Tono visual
Neón / bar nocturno. Logo y referencias por confirmar.

## Integraciones
Ninguna en foundation. Adapters se agregan cuando aparezca la necesidad real (auth, pagos, mensajería).

## Ambientes
Sólo `dev` por ahora. Producción pendiente.

## Estado
Foundation técnica completa. Páginas finales y features pendientes de cerrar el Brand Handshake.
