# Multilingual SEO policy

- Default locale: `es`. Secondary locale: `en`. (Confirmed during intake.)
- URL strategy is **not yet decided** (`/es` `/en` vs domain-based). Foundation does NOT activate `[lang]` routing.
- When activated, each translated route gets its own canonical and consistent `hreflang`.
- The sitemap reflects only approved translations. Incomplete translations are noindex.
- Copy is reviewed per locale (no literal translations on SEO surfaces).
